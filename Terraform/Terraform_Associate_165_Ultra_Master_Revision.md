# Critical Edge Case Reminders

-   State tracks managed resources, not provider configuration.
-   Sensitive hides CLI output, not state storage.
-   Remote backend provides locking; local does not.
-   Plan previews changes; apply executes them.
-   File order does not determine execution order.
-   Modules communicate via inputs and outputs only.
-   Import maps existing resources to state; it does not recreate them.
-   Version pinning ensures reproducibility.
-   Drift is detected during plan.
-   Always prefer declarative configuration changes over CLI shortcuts.
